# LiteBlock
Chrome extension for batch block users on Twitter(x.com)

# LiteBlock · X
**文艺清新风格的 X.com（Twitter）批量用户屏蔽插件**
一个界面淡雅、操作简便的 Chrome 浏览器扩展，帮助用户在 X 平台快速批量屏蔽指定用户。

![LiteBlock界面预览](https://via.placeholder.com/600x400/ f5f0eb/5f8a6f?text=LiteBlock·X)

---
## ✨ 功能特点
- **文艺清新界面**：采用莫兰迪色系 + 水墨风设计，字体优雅，视觉舒适
- **双模式切换**：
  - **用户名模式**：通过用户名、显示名称快速匹配并批量屏蔽
  - **发帖内容模式**：通过帖子内容关键词匹配并批量屏蔽对应用户
- **智能多格式输入**：支持英文逗号 `,`、中文顿号 `，`、换行符分隔多个条件
- **增强匹配机制**：同时匹配**显示名称**、**@用户名**、**用户简介**（Bio）
- **自动执行流程**：自动点击「更多」→ 「屏蔽」→ 「确认」完整流程
- **实时日志输出**：控制台详细显示检测过程，便于调试
- **仅限 X 平台**：精准适配 x.com / twitter.com，减少误触发

## 📥 安装方法
1. 下载或解压插件文件夹（文件夹名称建议为 `LiteBlock`）
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的 **「开发者模式」**
4. 点击左侧 **「加载已解压的扩展程序」**
5. 选择插件文件夹，点击确定即可加载

**图标**：
- `icons/icon16.png`
- `icons/icon48.png`
- `icons/icon128.png`
- `icons/icon256.png`

## 🚀 使用说明
### 基本操作流程
1. 打开 **X.com**（推荐在时间线、搜索结果页或单帖详情页使用）
2. 点击浏览器右上角的 **LiteBlock** 插件图标
3. 在弹出窗口中切换 Tab：
   - **用户名模式**（默认）：输入要屏蔽的用户名或关键词
   - **发帖内容模式**：输入帖子内容中的关键词
4. 在文本框中输入内容（支持多条）：用逗号、顿号或换行分隔
5. 点击 **「开始批量屏蔽」** 按钮
6. **保持当前 X 页面在前台**，不要切换标签页或最小化浏览器
7. 等待插件自动处理（会有合理延迟以降低风控风险）

### 输入示例
**用户名模式**：
- `赌博, 处男, 同城上门, 野战, 炮友`

**发帖内容模式**：
- `@6QOO0，@acrmrve，@BLACQKL_1`

---
## ⚠️ 注意事项

- **操作速度**：插件内置延迟以模拟人工操作，**建议单次处理不要超过 30 个用户**
- **页面状态**：必须保持 X 页面可见且在前台，滚动到需要处理的内容区域效果更佳
- **单帖页面**：在 `/status/` 页面同样可用，但内容较少时推荐使用时间线页面
- **风控提醒**：频繁大量操作可能触发 X 平台安全机制，建议分批次使用
- **权限范围**：插件仅在 `x.com` 和 `twitter.com` 生效
- **调试**：如未生效，请按 `F12` 打开控制台，查看以 `[LiteBlock]` 开头的日志

## 🔧 常见问题

**Q：点击按钮后显示“已尝试处理 0 个匹配用户”？**  
A：请检查输入内容是否与页面中的用户名/显示名/Bio 匹配。控制台会打印所有检测到的用户，可据此调整关键词。

**Q：菜单没有弹出或确认按钮没点击？**  
A：X 页面动态加载较频繁，建议刷新页面后重试，或稍等几秒再操作。

**Q：想调整操作速度？**  
A：目前延迟已优化，可在 `popup.js` 中修改 `setTimeout` 的毫秒数。
